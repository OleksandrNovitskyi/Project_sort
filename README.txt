# Project_sort
sort emploee

Робота програми зводиться до читання csv таблиці з даними по працівникам та очищення її по певним фільтрам.
Фільтри:
1. ім'я або прізвище працівника в "чорному списку". "Чорний список" формується вручну HR-ом та має дві колонки -
First_name і Last_name. Ім'я працівника в колонці 24, прізвище - в колонці 25.
2. в посаді працівника (колонка 43) наявне слово "NFT". слово можна замінити в тексті функції '''get_short_list''',
замінивши значення змінної '''forbidden_word'''
3. по фотографії з профілю (посилання в колонці 30) визначається вік та раса працівника за допомогою бібліотеки 
DeepFace. Інсталяція бібліотеки описана тут - https://github.com/serengil/deepface. Функція face_filter приймає
посилання на фотографію та налаштована на відсіювання працівників молодших 25 років та працівників не європеоїдів.
Для зміни віку та раси - достатньо додатково ввести два ввідні параметри при виклику функції
'''face_filter(row[30])'''(строка 21) на такі наприклад '''face_filter(row[30], 35, 'asian')''',
де 35 - нове обмеження по віку, 'asian' - монголоїдна раса. 
Перелік можливих рас - ['asian', 'indian', 'black', 'white', 'middle eastern', 'latino hispanic']

Результатом роботи програми є файл result.csv

Для перевірки працездатності програми - написано тест, який перевіряє відмінності між вхідним і вихідним файлами.
